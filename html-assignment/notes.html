HTML Notes

Here's some notes I've gathered concerning HTML tags

Tag Title Description
<p> Paragraph Isolates the text inside the tag from the rest of the text by adding a blank line on the top and bottom
<b> Break Causes the text inside the tag to be placed on a line below the rest of the text
<hr> Horizontal Rule Puts a line on the webpage used to separate portions of the page
<h1> Headline One Formats text inside the tag into the first and most important type of headline
<img> Image Displays images on the webpage when a source and alternative text is added into the tag
